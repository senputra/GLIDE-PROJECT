== Description ==

Add Flickr widgets, Instagram widgets, Social links widgets, gallery shortcodes and more on your theme.

= Enhanced gallery =

Brand Extra enhances the default WordPress gallery. It displays it in a grid (up to 9 columns) with a zoom effect on hover.

When click on an image it will be open in a slider.

You can choose a custom button to display on each image and a custom url to open on click.

= Instagram widget =

Brand Extra provides an Instagram widget to add in one of your theme widgets areas.

Search the *Instagram Feed* widget, add a title, the username, the number of photo that you want to show (up to 12), and the number of columns.

= Flickr widget =

Brand Extra provides a Flickr widget to add in one of your theme widgets areas.

Search the *Flickr Stream* widget, add a title, the username id and the number of photo that you want to show (up to 10).

= Social links widget =

Brand Extra provides a Social links widget to add links to your social page in one of your theme widgets areas.

Search the *Social links* widget, choose the size of the icons, the shape (circle, square or none), the alignment.

The *Social links* widget supports the following social apps:
* facebook
* twitter
* google
* instagram
* pinterest
* flickr
* 500px
* github
* linkedin
* tumblr
* vimeo
* youtube

== Installation ==

**From your WordPress dashboard**

1. Visit 'Plugins -> Add New'
2. Click on 'Upload Plugin'
3. Select the brand-extra zip file you have downloaded
4. Click on 'Install Now'
5. Activate 'Brand Extra' from your plugins page.
